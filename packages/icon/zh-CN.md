# kn-icon

### 使用指南

```javascript
import KnIcon from 'kn-ui'

Vue.use(KnIcon)
```

### 代码演示

#### 基础用法

```html
<kn-icon />
```

### Cell API

| 参数      | 说明     | 类型     | 默认值   | 版本 |
| --------- | -------- | -------- | -------- | ---- |
| icon      | 图标类型 | `String` | -        | -    |
| icon-size | 图标大小 | `String` | `normal` | -    |

### Cell Event

| 事件名 | 说明           | 参数 |
| ------ | -------------- | ---- |
| click  | 点击图标时触发 | -    |
